# ForgeIT WireMock Fixtures

This folder holds WireMock request/response payloads used by ForgeIT tests. Add JSON fixtures under:
- `forge-it/wiremock/request` for request bodies
- `forge-it/wiremock/response` for response bodies
- `forge-it/wiremock/default/request` and `forge-it/wiremock/default/response` for reusable defaults

See project ForgeIT documentation for builder options and default resolution.
